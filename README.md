# Niranjan Jewellery Website

A PHP-based jewellery e-commerce website built as a college project.

## Features
- Product catalogue with cart functionality
- User registration and login (session-based)
- Responsive design with elegant gold theme
- Server-side form validation

## Tech Stack
- Frontend: HTML5, CSS3
- Backend: PHP 8+
- Storage: JSON (flat-file for users), PHP Sessions (cart)

## How to Run
1. Install XAMPP or any PHP server
2. Place the folder in `htdocs/`
3. Open `http://localhost/niranjan-jewellery/`
